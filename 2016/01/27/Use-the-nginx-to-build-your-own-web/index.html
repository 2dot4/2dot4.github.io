Use the nginx to build your own web

發表於 2016-01-27

安装Hexo

1 $ sudo npm install -g hexo

注意，要加sudo，否则可能提示无权限访问。(npm: node package manager)

卸载Hexo方法：

1 terminal至/usr/local/lib
2 $ sudo npm uninstall hexo

启用Hexo

  1. 新建目标文件夹，如git_blog
  2. terminal至git_blog

1 $ hexo init // Hexo会在git_blog中建立网站所需要的文件
2 $ npm install // ** 注意，一定要加这个命令！！！！否则生成的public\index.html文件可能各种空白、错误。**
3 $ hexo g // 生成网页
$ hexo s // 开启server
4 $ hexo s // 开启server

打开网址：http://localhost:4000/ ，可以看到HelloWorld的网页后，说明安装成功

设置Hexo

设置_config.yml文件

编辑Hexo安装文件夹中的_config.yml文件，不再赘述
如果是要搭到Github Pages上的话，需要
1 deploy:
2 type: github
3 repository: git@github.com:vin-zhou/vin-zhou.github.com.git
4 branch: master

更换主题

这里使用了jacman主题
1 $ git clone https://github.com/wuchong/jacman.git themes/jacman
相应已修改了_config.yml中的theme值。

更新主题

1 $ cd themes/jacman
2 $ git pull

  2. 在footer.ejs中， 末尾添加js代码：
